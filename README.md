# 🚀 SmartSeat Portal | Ultimate Exam Management System

A premium, role-based exam seating and student welfare management platform. Built for efficiency, transparency, and advanced administrative control.

---

## ✨ Key Features

### 👨‍🎓 Student Portal
- **Interactive Seating Finder:** Real-time lookup of exam hall and seat assignments.
- **Dynamic Timetable:** Personalized exam schedules and resource center.
- **Assignments & Grades:** Integrated view of student tasks and CAT-2/CAT-3 performance.
- **Support System:** Help and FAQ section for exam-related queries.

### 🛡️ Admin & Staff Tiers
- **Smart Seating Generator:** Algorithmic hall allocation based on student data and capacity.
- **Interactive Map View:** Real-time visualization of exam hall seating status.
- **Student Welfare Hub:** Integrated assignment tracking and bulk notification system.
- **Data Insights:** Live analytics on student attendance and exam utilization.
- **Backup & Recovery:** Robust system for data persistence and restoration.

---

## 🛠️ Tech Stack

- **Frontend:** React 19 + Vite
- **Logic:** Context API for Global State Management
- **Icons:** Lucide-React
- **Visualization:** Recharts & Three.js 
- **Styling:** Custom CSS (Glassmorphism & Neon Aesthetics)

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/YOUR_USERNAME/smartseat-portal.git
cd smartseat-portal
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run Development Server
```bash
npm run dev
```
Open `http://localhost:5173` to view the portal.
